# game-scope-profile

Part of the **Game Superpowers** skill system.

## Summary

Set the game scope tier, completeness bar, and allowed refactor range so the agent neither underbuilds nor gets lost in endless overbuilding.

## Recommended usage

- Use `game-scope-profile` for this game project and keep the output evidence-based.
- Apply `game-scope-profile` to improve or assess an existing browser game in a focused way.
- Use `game-scope-profile` as part of a larger Game Superpowers workflow when the request clearly matches game scope profile.
