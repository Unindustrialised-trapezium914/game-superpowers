# using-game-superpowers

Part of the **Game Superpowers** skill system.

## Summary

Use when the task is about building, auditing, or improving a game project with this Game Superpowers skill collection. Acts as the collection bootstrap skill: classify the request, choose the right specialized game skills, and coordinate them instead of trying to solve the entire task ad hoc.

## Recommended usage

- Use `using-game-superpowers` for this game project and keep the output evidence-based.
- Apply `using-game-superpowers` to improve or assess an existing browser game in a focused way.
- Use `using-game-superpowers` as part of a larger Game Superpowers workflow when the request clearly matches using game superpowers.
