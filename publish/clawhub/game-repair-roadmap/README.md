## Summary

- Use `game-repair-roadmap` for this game project and keep the output evidence-based.
- Apply `game-repair-roadmap` to improve or assess an existing browser game in a focused way.
- Use `game-repair-roadmap` as part of a larger Game Superpowers workflow when the request clearly matches game repair roadmap.
