# game-ux-flow-designer

Part of the **Game Superpowers** skill system.

## Summary

Design the interaction and interface flow before or alongside implementation. Focus on the first minutes, clarity, menus, onboarding, and feedback so the game feels understandable and intentional.

## Recommended usage

- Use `game-ux-flow-designer` for this game project and keep the output evidence-based.
- Apply `game-ux-flow-designer` to improve or assess an existing browser game in a focused way.
- Use `game-ux-flow-designer` as part of a larger Game Superpowers workflow when the request clearly matches game ux flow designer.
