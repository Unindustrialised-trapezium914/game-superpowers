# gsp-project-state-assessment

Part of the **Game Superpowers** skill system.

## Summary

Use before major game work to classify the repo as greenfield, prelaunch, shipped, or live-risky.

## Recommended usage

- Use `gsp-project-state-assessment` for this game project and keep the output evidence-based.
- Apply `gsp-project-state-assessment` to improve or assess an existing browser game in a focused way.
- Use `gsp-project-state-assessment` as part of a larger Game Superpowers workflow when the request clearly matches game project state assessment.
