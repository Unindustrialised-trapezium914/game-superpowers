# First-Playable Spec Template

This template is now a **specialized subset** of the broader build brief.
Use it only when the quality target is explicitly `first-playable`.

## IntentSpec
- **One-sentence goal**
- **Primary fantasy**
- **Must-have constraints**
- **Non-goals**

## ExperienceSpec
- **First playable goal**
- **First 30 seconds**
- **Primary verb**
- **Failure loop**
- **HUD priorities**
- **Input expectations**

## GameSystemSpec
- **Core loop**
- **State model**
- **Player controller model**
- **Camera model**

## VisualRecipeSpec
- **Style direction**
- **Procedural art level**
- **UI theme recipe**

## BackendProfileSpec
- **Capability needs**
- **Compare mode needed?**

## EvalSpec
- **Must boot**
- **Must reach actionable screen**
- **Must execute primary verb**
- **Must display readable HUD**
- **Must handle fail/retry**
