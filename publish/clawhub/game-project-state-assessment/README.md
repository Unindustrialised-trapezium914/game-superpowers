# game-project-state-assessment

Part of the **Game Superpowers** skill system.

## Summary

Use before major game work begins. Inspect the repo and request to classify the project as greenfield, prelaunch iteration, shipped product, or live-risky work. This determines whether aggressive AI-native development is appropriate.

## Recommended usage

- Use `game-project-state-assessment` for this game project and keep the output evidence-based.
- Apply `game-project-state-assessment` to improve or assess an existing browser game in a focused way.
- Use `game-project-state-assessment` as part of a larger Game Superpowers workflow when the request clearly matches game project state assessment.
