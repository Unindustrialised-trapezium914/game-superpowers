# game-implementation-plan

Part of the **Game Superpowers** skill system.

## Summary

Turn the approved brief into a sequence of verifiable tasks that matches the selected development mode. This skill prevents both endless micro-edits and reckless unbounded rewrites.

## Recommended usage

- Use `game-implementation-plan` for this game project and keep the output evidence-based.
- Apply `game-implementation-plan` to improve or assess an existing browser game in a focused way.
- Use `game-implementation-plan` as part of a larger Game Superpowers workflow when the request clearly matches game implementation plan.
