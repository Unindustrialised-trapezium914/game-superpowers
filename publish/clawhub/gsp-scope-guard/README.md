# gsp-scope-guard

Part of the **Game Superpowers** skill system.

## Summary

Use when a game project is expanding too broadly and needs scope protection without losing the core goal.

## Recommended usage

- Use `gsp-scope-guard` for this game project and keep the output evidence-based.
- Apply `gsp-scope-guard` to improve or assess an existing browser game in a focused way.
- Use `gsp-scope-guard` as part of a larger Game Superpowers workflow when the request clearly matches game scope guard.
