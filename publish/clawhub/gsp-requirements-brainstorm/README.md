# gsp-requirements-brainstorm

Part of the **Game Superpowers** skill system.

## Summary

Use when a game request is vague, contradictory, or under-specified and needs a production-ready brief.

## Recommended usage

- Use `gsp-requirements-brainstorm` for this game project and keep the output evidence-based.
- Apply `gsp-requirements-brainstorm` to improve or assess an existing browser game in a focused way.
- Use `gsp-requirements-brainstorm` as part of a larger Game Superpowers workflow when the request clearly matches game requirements brainstorm.
