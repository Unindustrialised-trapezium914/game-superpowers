# game-requirements-brainstorm

Part of the **Game Superpowers** skill system.

## Summary

Use when the request is vague, contradictory, or too under-specified to build a strong game result. Turn the request into a concise production-ready requirement brief without getting stuck in endless ideation.

## Recommended usage

- Use `game-requirements-brainstorm` for this game project and keep the output evidence-based.
- Apply `game-requirements-brainstorm` to improve or assess an existing browser game in a focused way.
- Use `game-requirements-brainstorm` as part of a larger Game Superpowers workflow when the request clearly matches game requirements brainstorm.
