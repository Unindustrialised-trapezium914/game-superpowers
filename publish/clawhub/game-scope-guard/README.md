# game-scope-guard

Part of the **Game Superpowers** skill system.

## Summary

Protect the project from runaway complexity without destroying the user’s real goal. Scope differently depending on whether the project is greenfield, refactor-friendly, or live-risky.

## Recommended usage

- Use `game-scope-guard` for this game project and keep the output evidence-based.
- Apply `game-scope-guard` to improve or assess an existing browser game in a focused way.
- Use `game-scope-guard` as part of a larger Game Superpowers workflow when the request clearly matches game scope guard.
