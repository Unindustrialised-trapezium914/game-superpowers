# game-rolling-supervisor

Part of the **Game Superpowers** skill system.

## Summary

Enable a guarded external rolling build/verify/repair loop for high-autonomy game development. Use when the task is too large for one clean session and broad changes are allowed.

## Recommended usage

- Use `game-rolling-supervisor` for this game project and keep the output evidence-based.
- Apply `game-rolling-supervisor` to improve or assess an existing browser game in a focused way.
- Use `game-rolling-supervisor` as part of a larger Game Superpowers workflow when the request clearly matches game rolling supervisor.
