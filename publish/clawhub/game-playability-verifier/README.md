# game-playability-verifier

Part of the **Game Superpowers** skill system.

## Summary

Verify the build against the chosen quality target. Check bootability, interaction, UX, requested feature completeness, and regressions before success is claimed.

## Recommended usage

- Use `game-playability-verifier` for this game project and keep the output evidence-based.
- Apply `game-playability-verifier` to improve or assess an existing browser game in a focused way.
- Use `game-playability-verifier` as part of a larger Game Superpowers workflow when the request clearly matches game playability verifier.
