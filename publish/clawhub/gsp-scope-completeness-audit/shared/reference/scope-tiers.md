# Scope tiers

## micro
Mechanic spike or proof of concept.
Keep only the core interaction and minimal surrounding structure.

## small
Narrow but coherent build.
One core loop, one or two states, enough UI and feedback to evaluate honestly.

## medium
Strong prototype scope.
Core loop, meta flow, retry / fail flow, enough content and feedback to feel intentional.

## ambitious
Large coherent target that can still be delivered in AI-native chunks.
Requires sharp deferrals and usually benefits from the rolling supervisor.
