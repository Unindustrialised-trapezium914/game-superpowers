# gsp-concept-brainstorm

Part of the **Game Superpowers** skill system.

## Summary

Use when a game idea is still vague and needs pillars, verbs, and an acceptance bar before implementation.

## Recommended usage

- Use `gsp-concept-brainstorm` for this game project and keep the output evidence-based.
- Apply `gsp-concept-brainstorm` to improve or assess an existing browser game in a focused way.
- Use `gsp-concept-brainstorm` as part of a larger Game Superpowers workflow when the request clearly matches game concept brainstorm.
