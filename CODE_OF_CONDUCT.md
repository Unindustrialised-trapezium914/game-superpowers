# Code of Conduct

## Our standard

This project aims to be open, practical, and respectful.

People participating in this repository are expected to:

- be respectful in discussion and review
- assume good intent, while still being technically rigorous
- give concrete, actionable feedback
- stay on topic and keep collaboration productive
- welcome different experience levels and backgrounds

Unacceptable behavior includes:

- harassment, intimidation, or personal attacks
- discriminatory or exclusionary language
- bad-faith trolling or repeated derailment
- publishing private information without permission
- hostile conduct that makes collaboration harder or unsafe

## Scope

This Code of Conduct applies to:

- issues
- pull requests
- discussions about this repository
- other project spaces directly tied to this repository

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or other contributions that do not align with this Code of Conduct.

Maintainers may also temporarily or permanently limit participation for repeated or severe violations.

## Reporting

If you encounter unacceptable behavior, open a private channel with the maintainers if possible.

If no private reporting path is available yet, open an issue only if you can do so safely without exposing someone to harassment or publishing private information.

## Maintainer guidance

When enforcing this policy, prefer:

- de-escalation first
- clear explanations
- the minimum action needed to restore a productive environment

The goal is to protect contributors and keep the project usable.
